1. 기능 소개

이 페이지 에서는 팀 관리 API를 소개 합니다.
팀 관리 API는 팀을 생성, 삭제, 수정 할 수 있을 뿐만 아니라 팀원을 편집할 수 있습니다.
팀 관리 API는 REST API로 구현이 되었으며 팀 관리 페이지에서 손쉽게 사용할 수 있습니다.
팀 관리 관련 API를 사용하기 위해서는 Header에 Bearer Token을 포함 해야 합니다.

2. 팀 생성

팀 추가 버튼을 누르면 다음의 모달 창이 뜹니다.

모달 창의 모든 항목을 입력하고 저장 버튼을 누르면 팀이 추가 됩니다.

팀이 등록이 완료 되면 모달이 닫히고 추가된 팀을 볼 수 있습니다.

3. 팀 조회

팀 목록에서 각 팀마다 +detail 버튼이 있습니다.
이 버튼을 누르면 각 팀을 자세히 조회할 수 있습니다.

팀 정보중 Client id를 제외한 모든 항목은 수정이 됩니다.

4. 팀 명/서비스 명 수정

팀 이름을 확인 할 수 있는데 옆의 토글 버튼을 누르면 팀 이름 수정 폼이 열린다.
수정할 이름을 입력하고 '수정'버튼을 누르면 팀 이름이 수정된다.

서비스 명을 확인할 수 있는데 수정 버튼을 누르면 서비스 명을 수정할 수 있는 모달이 뜬다.

새로운 서비스 명을 입력하고 수정 버튼을 누르면 서비스 명이 수정이 된다.

5. 리다이렉트 URL 추가/삭제

리다이렉트 url 입력 폼이 보이고 그 밑에 등록된 url 목록이 보인다.

새 리다이렉트 url을 입력하고 추가 버튼을 누르면 새 리다이렉트 url이 등록된다.

리다이렉트 url이 추가되면 아래에 추가된 url이 보인다.

리다이렉트 url은 x 표시를 누르면 삭제된다.

6. 팀원 추가/삭제

팀원을 확인할 수 있다.

팀원으로 추가할 개발자를 검색할수 있고 검색한 이메일을 넣고 추가 버튼을 누르면 팀 멤버로 추가가 된다.

팀장을 제외한 팀원은 x를 누르면 팀원에서 제외된다.

7. 팀 이미지 수정/삭제

팀 이미지를 확인 할 수 있습니다.
수정을 누르면 이미지를 수정할 수 있는 모달이 뜹니다.

업로드 할 이미지를 넣고 수정 버튼을 누르면 수정이 됩니다.

삭제 버튼을 누르면 이미지가 초기화 됩니다.
